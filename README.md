## Installation
``` r
devtools::install_github("kongdd/Ipaper")
```

* Batch download paper with doi.

* Base tools, e.g. `dir.show`, `runningId`, `subl` and `github`

* Parallel functions, e.g. `InitCluster`, `killCluster`.
